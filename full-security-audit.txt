1. INTRODUCTION
================================================================================

Complete compromise of a popular online code compilation platform
through multiple critical attack vectors leading to full source code leak
and infrastructure compromise.


================================================================================
2. ATTACK VECTOR & EXPLOITATION CHAIN
================================================================================

2.1 Initial Reconnaissance
--------------------------
- Opened the target site's online compiler page
- Inspected page source, JavaScript files, and network requests
- Found API keys hardcoded in client-side JavaScript
- Found weak socket.io authentication (only a single parameter)
- Identified prototype pollution risks via jQuery.extend

2.2 Server-Side Code Execution (RCE)
-------------------------------------
- The platform's Python compiler has NO proper sandboxing
- Executed Python code using os, subprocess, socket modules
- Achieved full command execution on the Docker container

2.3 Cloud Metadata Exploitation (SSRF)
---------------------------------------
- Accessed cloud provider metadata endpoint
- Retrieved full instance metadata
- Retrieved cloud service account OAuth2 tokens
- Retrieved SSH public keys and startup/shutdown scripts

2.4 Source Code Repository Access
----------------------------------
- Startup script contained VCS OAuth token
- Full access to the source code repository
- All server-side code, config files, and infrastructure scripts

2.5 SSL/TLS Private Key Exposure
----------------------------------
- TLS keys stored in source repository
- Full RSA private key for HTTPS
